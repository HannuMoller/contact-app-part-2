- Windows 7

    - MariaDB
  - Microsoft Visual Studio
  - IntelliJ IDEA
  
## Environments

### Local Storage

In _IntelliJ IDEA_ in it's Terminal window execute command:

    ng serve --environment=local
    
This allows definitions in file _environment.local.ts_ to be used.
